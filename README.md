# Scene Structure

- Managers
    - Manage state of the game
- Setup (essential objects that need to persist throughout the game.)
    - Camera
    - Light
    - Event
- Environment
    - Terrain
- Units
    - Enemy
    - Player
- Canvases
    - canvas (UI)
- Systems (don't destroy on load = persists across scene changes)
    - Audio System
    - Resource System
